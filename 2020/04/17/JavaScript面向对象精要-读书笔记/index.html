第一章

基本类型和引用类型

1.1什么是类型

原始类型保存为简单的数据值，引用类型保存为对象，其本质是指向内存位置的引用。
为了让开发者能够把原始类型和引用类型按照相同的方式处理（为啥要用相同的方式？）js花费了很大的力气来保证语言的一致性。
其他编程语言用stack存储原始类型，用heap存储引用类型。
js则不同：它用一个变量对象追踪变量的生命周期（原始值被直接保存在变量对象内，而引用值则作为一个指针保存在变量对象内，也就是说大家都保存在变量对象内）

1.2原始类型

boolean,number,string,null,undefined
js和其他语言一样，原始类型的变量直接存储原始值（引用类型存的是内存中的存储位置）

鉴别原始类型

typeof ‘hfskjdhf’ // string
typeof 10 // number
typeof true // boolean
typeof undefined // undefined
typeof null // object empty object point
use value === null // true or false to
console.log(“5” == 5); // true
console.log(“5” === 5); // false
== 会转换类型 ===则不会

1.9 总结

javascript虽然没有类，但是有类型。每个变量或数据都有一个对应的原始类型或者引用类型。5种原始类型（Number String Boolean null undefined）的值会直接保存在变量对象中，除了null,都可以用typeof来甄别。null用value===null来甄别。
引用类型是javascript中最接近类的东西，而对象是引用类型的实例。可以用new操作符或字面量形式创建新的对象。通常可用[].访问属性和方法。函数在js中也是对爱哪个，可以用typeof来甄别他们。至于其他引用类型，可以用instanceof来鉴别。
为了让原始类型看上去更像是引用类型（为什么要让原始类型看上去要想引用类型？），js提供了3中原始封装类型：String Number Boolean。javascript会在背后创建这些对象，使得你可以像使用普通对象一样使用原始类型。但这些临时对象在使用他们的语句结束时，就立即被销毁。

第二章

函数

函数其实就是对象，函数不同于其他对象的一点是函数存在一个被称为call的内部属性。内部属性无法通过代码访问，而是定义了代码执行时的行为。js对象中有多种内部属性，这些内部属性都用[[]]来标注。
[[call]]是funcuton独有的，表明该对象可以被执行。由于仅fucntion有这个属性，ECMASctipt定义typeof操作符对任何具有[[call]]属性的对象返回’function’
